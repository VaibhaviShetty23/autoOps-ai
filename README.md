# AutoOps AI (Enterprise)

Enterprise-style multi-agent AI system for automated incident detection, root cause analysis, and remediation.

## Features
- Multi-agent architecture
- Gemini LLM integration
- Incident memory bank
- Anomaly detection
- Log analysis
- Agent evaluation
- Observability (logs + tracing)

## Run the Project
```bash
pip install -r requirements.txt
python main.py
